# calculator-master 工程文件描述

## 🔧 核心文件解析

### 1. **Index.ets / CardCalc.ets** (计算器核心逻辑)

**位置**: `entry/src/main/ets/pages/Index.ets` 和 `entry/src/main/ets/calc/pages/CardCalc.ets`

**功能**:
- 实现完整的计算器UI和计算逻辑
- 使用声明式UI构建界面（Column、Row、Button、Image组件）

**核心算法**:

| 函数 | 功能 |
|------|------|
| `calc(inputContent)` | 主入口：中缀→后缀→计算结果 |
| `parseInfixExpression()` | 解析中缀表达式，拆分数字和运算符 |
| `toSuffixExpression()` | 中缀转后缀（逆波兰表达式），使用栈结构 |
| `calcSuffixExpression()` | 计算后缀表达式，使用数字栈 |
| `getFloatNum()` | 处理浮点数精度，动态计算小数位数 |
| `isOperator()` | 判断是否为运算符 (+ - * /) |
| `isPrioritized()` | 比较运算符优先级 (* / > + -) |

**计算示例**:
```
输入: "3+4*2"
↓ parseInfixExpression
中缀: ["3", "+", "4", "*", "2"]
↓ toSuffixExpression
后缀: ["3", "4", "2", "*", "+"]
↓ calcSuffixExpression
结果: "11"
```

**UI布局**:
```
┌─────────────────────┐
│   表达式显示区      │
├─────────────────────┤
│ [C] [/] [*] [⌫]    │
│ [7] [8] [9] [-]     │
│ [4] [5] [6] [+]     │
│ [1] [2] [3] [0]     │
│      [=]            │
└─────────────────────┘
```

### 2. **EntryAbility.ets** (应用入口)

**位置**: `entry/src/main/ets/entryability/EntryAbility.ets`

**功能**:
- 继承 `UIAbility`，管理应用生命周期
- 加载主页面: `windowStage.loadContent('pages/Index')`

**生命周期方法**:
```
onCreate() → onWindowStageCreate() → onForeground()
                                    ↓
                              加载Index页面
                                    ↓
onBackground() → onWindowStageDestroy() → onDestroy()
```

---

### 3. **Logger.ts** (日志工具)

**位置**: `entry/src/main/ets/model/Logger.ts`

**功能**:
- 封装 `hilog` 日志接口
- 提供四个日志级别: `debug` / `info` / `warn` / `error`
- 使用域 `0xFF00`，格式: `%{public}s, %{public}s`

## 🎯 技术要点

1. **算法设计**: 使用逆波兰表达式（后缀表达式）计算，避免括号和优先级问题
2. **UI框架**: ArkTS声明式UI，组件化开发
3. **状态管理**: `@State` 响应式变量驱动界面更新
4. **精度处理**: 动态计算小数位数，避免浮点数精度丢失
5. **生命周期**: 完整的Ability生命周期管理
6. **国际化**: 支持中英文多语言
