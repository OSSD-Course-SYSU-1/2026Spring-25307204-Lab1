# 跨平台科学计算器工程文件描述文档

## 一、项目概览

### 1.1 基本信息
- **项目名称**：跨平台科学计算器
- **项目标识**：com.samples.arktscalc
- **项目类型**：HarmonyOS应用
- **版本信息**：v1.0.0 (versionCode: 1000000)
- **技术栈**：
  - **框架**：ArkUI-X（跨平台声明式UI框架）
  - **语言**：ArkTS/TypeScript
  - **构建工具**：Hvigor
  - **测试框架**：Jest
  - **代码质量**：ESLint + TypeScript严格模式
- **目标平台**：HarmonyOS 5.0.5+、Web浏览器（Chrome 80+、Firefox 75+、Safari 13+、Edge 80+）

### 1.2 核心特性
- 基础计算：四则运算、运算符优先级、小数运算、除零处理
- 科学计算：平方根、对数、三角函数、反三角函数、指数运算、阶乘
- 跨平台支持：HarmonyOS原生、Web浏览器（PC/移动）、平板
- 响应式布局：自适应屏幕尺寸（320px-1920px），支持横竖屏切换
- 多输入方式：触控、键盘快捷键、触控笔（平板）
- 平板优化：双面板布局、高分辨率适配、60fps动画

### 3.1 业务逻辑层（common模块）

#### 3.1.1 计算引擎

**文件路径**：`common/src/main/ets/engine/CalculatorEngine.ets`

**功能说明**：
- 核心计算引擎，实现基于逆波兰表达式（RPN）的计算逻辑
- 支持四则运算（+、-、*、/）和括号运算
- 实现运算符优先级处理

**核心函数**：
```typescript
// 主入口函数：计算表达式
export function calc(inputContent: string): string

// 解析中缀表达式
function parseInfixExpression(inputContent: string): string[]

// 中缀表达式转后缀表达式（逆波兰表达式）
function toSuffixExpression(expression: string[]): string[]

// 计算后缀表达式
function calcSuffixExpression(expression: string[]): string[]
```

**运算符优先级定义**：
```typescript
const OPERATORLEVELS = {
  '+': 0,
  '-': 0,
  '*': 1,
  '/': 1
}
```

**特性**：
- 自动处理浮点数精度问题
- 支持负数运算
- 结果超过15位自动转为科学计数法

### 3.2 UI层（entry模块）

#### 3.2.1 主页面

**文件路径**：`entry/src/main/ets/pages/Index.ets`

**功能说明**：
- 计算器主页面UI
- 使用ArkUI声明式UI语法
- 实现按钮布局和交互逻辑

**组件结构**：
```typescript
@Entry(storage)
@Component
struct Index {
  @State result: string = ''
  @State expression: string = ''
  
  build() {
    Column() {
      // 显示屏区域
      Column() {
        Text(this.expression)
          .fontSize(28)
          .fontColor('#333333')
          .textAlign(TextAlign.End)
      }
      
      // 按钮区域（4行按钮 + 等号按钮）
      Column() {
        Row() { /* 第一行按钮：sqrt, C, /, *, 删除 */ }
        Row() { /* 第二行按钮：ln, 7, 8, 9, - */ }
        Row() { /* 第三行按钮：π, 4, 5, 6, + */ }
        Row() { /* 第四行按钮：1, 2, 3, 0, . */ }
        Row() { /* 等号按钮 */ }
      }
    }
  }
}
```

**输入处理逻辑**：
- 清除（C）：清空表达式和结果
- 删除：删除最后一个字符
- 科学函数（sqrt、ln）：计算结果并替换表达式
- π：插入圆周率值
- 运算符：处理运算符连续输入
- 等号（=）：计算最终结果
- 数字：追加到表达式并实时计算预览

## 四、技术实现分析

### 4.1 计算引擎实现

**算法原理**：逆波兰表达式算法

**执行流程**：
```
输入表达式 → 解析中缀表达式 → 转换为后缀表达式 → 计算后缀表达式 → 输出结果
```

**关键技术点**：
1. 中缀表达式解析：识别数字、运算符、括号
2. 逆波兰转换：使用栈结构处理运算符优先级
3. 表达式求值：使用栈结构存储操作数
4. 精度处理：浮点数运算精度控制

---

### 4.2 状态管理实现

**设计模式**：观察者模式

**实现原理**：
- 单一状态源（CalculatorState）
- 订阅者列表管理
- 状态更新时自动通知所有订阅者

---

### 4.3 分布式能力实现

**卡片功能**：
- 继承FormExtensionAbility
